Overview

The okapi-olifant distribution contains Olifant, a cross-platform application to create and maintain translation memory repositories. It allows you to work primarily with TMX files but can also work with other formats such such as PO, XLIFF, and other translation formats.

The source code is available in the okapi-olifant Google Code project.

For a list of changes since the previous release, please see the Changes Log document.

The latest version of the help is available on-line on the Okapi Framework Web site.

Windows Platform

Requirements

In order to run Olifant on Windows you must have a running installation of Java 1.6 or later. Sun Java, or an OpenJDK VM is recommended.

Installation

To install Olifant on Windows (32- and 64-bit):

  1. Download the ZIP file of the Windows distribution.
  2. Unzip the file on any location, we recommend C:\Program Files\Olifant.

To run Olifant: simply double-click on the olifant.exe file in the main folder.

You can also run the application with the following command line (assuming it has been unzipped in the recommended folder):

C:\>javaw -jar C:\Program Files\Olifant\lib\rainbow.jar

Uninstallation

To uninstall:

  1. Olifant runs independently of Windows, so to uninstall it you can simply delete the folder where you have installed it. There is no Registry information to remove.
  2. If you want you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Olifant.

Linux Platforms

Requirements

In order to run Olifant on Linux you must have a running installation of Java 1.6 or later. Sun Java, or an OpenJDK VM is recommended.

Installation

To install Olifant on Linux (32- and 64-bit):

  1. Download the ZIP file of the Linux distribution you need (there is one for 32-bit and one for 64-bit).
  2. Extract the file on any location. Make sure to preserve the full paths when extracting.

To runOlifant: simply double-click on the olifant.sh file in the main directory.

You can also run the application from the command line (assuming Olifant is installed in \home\archie\olifant):

archie@archie:~>java -jar okapi/lib/olifant.jar

Uninstallation

To uninstall:

  1. Delete the directory where Olifant is installed.
  2. If you want, you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Olifant.

Macintosh Platform

Requirements

In order to run Olifant on Macintsoh you must have Java 1.6 installed. Note that Apple does not provide Java 1.6 for all versions and architectures of the Mac.

  • If you have a 32-bit machine, you can run Java 1.6 only on Mac OS X 10.6 (Snow Leopard) using the 32-bit mode.
  • If you have a 64-bit machine, you can run Java 1.6 on either Mac OS X 10.5 (Leopard) or Mac OS X 10.6 (Snow Leopard) using the 64-bit mode. On Mac OX 10.7 (Lion) there is no pre-installed Java Runtime, but it can be installed easily.

Installation

To install Olifant on Macintosh (32- and 64-bit):

  1. Download the ZIP file of the Macintosh distribution you need (See above).
  2. Extract the ZIP file on any location. For example /Users/admin/Desktop/Olifant.

To run Olifant:

  1. The main directory of the distribution contains a Olifant application bundle (Olifant.app): simply double-click on it.
  2. If you run the application for the first time since you downloaded it, you may be prompted to confirm you want to run it.

You can modify the shell file olifant.sh that is in the /Contents/MacOS folder of the application bundle to fit your own parameters. In all cases you MUST specify the -XstartOnFirstThread option or the application may not work properly. You can make sure to run the Java VM in a specific mode using the -d32 and -d64 flags.

Uninstallation

To uninstall:

  1. Delete the directory where Olifant is installed.
  2. If you want, you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Olifant.
