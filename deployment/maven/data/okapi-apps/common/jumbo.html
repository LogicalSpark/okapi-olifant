Help - Okapi Jumbo @version@, @date@

Jumbo is a shell program that allows you to execute from a command-line prompt many of the operations you can perform with Olifant.

You can also execute one or more commands from the command-line that starts Jumbo, allowing you to run the tool in batch mode.

For example:

C:\>jumbo -c "use default; open myTM; add field someNewField; quit"

Commands

use[ default]
Opens the local default repository. It creates it if there is no existing one.
The local default repository is named defaultOlifantTMRepository and is located in the Olifant sub-directory of your home directory.
 
use memory
Opens an in-memory repository.
 
open <tmName>
Opens the TM <tmName>. This TM becomes the current TM. You can have several TMs open at the same time.
To switch to a TM already opened, just do an open <tmName> command again.
 
close[ <tmName>]
Closes the TM named <tmName>. If the parameter is not provided, the current TM is closed.
 
close all
Closes all open TMs.
 
create <tmName>[ <srcLoc>]
Creates a new TM in the repository. If <srcLoc> is not specified EN is used. If that TM exists already, it is simply opened. The TM becomes the current TM.
 
add locale <locale>
Adds the locale <locale> to the current TM.
 
add field <fieldName>
Adds the field <fieldName> to the current TM.
 
delete locale <locale>
Deletes the locale <locale> from the current TM.
 
delete field <fieldName>
Deletes the field <filedName> from the current TM.
 
quit|exit
Exits the application.
 
show tms
Lists all the TMs in the current repository.
 
show columns
Displays the list of the fields in the current TM, that are shown when listing entries. To list all the fields of the TM use show fields.
 
show fields
Displays the list of the fields available in the current TM.
 
show locales
Displays the list of the locales in the current TM.

Parameters

The following parameters can be used from the Jumbo's command-line:

  • -info : Displays the version and other information about the tool.
  • -c "<commands>" : Executes the given set of commands. Use the commands exit or quit to exit Jumbo at the end of the execution.
