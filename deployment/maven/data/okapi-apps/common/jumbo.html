Okapi Jumbo

Jumbo is a shell program that allows you to execute from a command-line prompt many of the operations you can perform with Olifant.

Commands

use[ default]

Opens the local default repository. It creates it if there is no existing one.

use memory

Opens an in-memory repository.

open <tmName>[ <srcLoc>[ <trgLoc>]]

Opens the TM <tmName>. This TM becomes the current TM. You can have several TMs open at the same time. To switch TM, just to an open <tmName> command again.

close[ <tmName>]

Closes the TM named <tmName>. If the parameter is not provided, the current TM (if there is one) is closed.

close all

Closes all open TMs.

create <tmName>[ <srcLoc>]

Creates a new TM in the repository. If that TM exists already, it is simply opened. The TM becomes the current TM.

add locale <locale>
add field <fieldName>
remove locale <locale>
remove field <fieldName>
set source <srcLoc>
set target <trgLoc>
<alias> is <name>
quit|exit

Exit the applications.

show tms

Lists all the TMs in the current repository (if one is active).

show columns

Displays the list of the fields in the current TM, that are shown when listing entries. To list all the fields of the TM use show fields.

show fields

Displays the list of the fields available in the current TM.

show locales

Displays the list of the locales in the current TM.
