Okapi Jumbo

Jumbo is a shell program that allows you to execute from a command-line prompt many of the operations you can perform with Olifant.

<No active repository> J>use default
repository: defaultOlifantLocalRepository
TMs in the repository:
- test1
- test2

<no active TM> J>open test2

test2 J>show locales
"EN", "FR"

test2 J>

Commands

use[ memory|default|<path>]

Sets the repository to work with. If no parameter is given, the default is default.

use memory

Opens a in-memory repository.

use default

Opens the local default repository. It creates it if there is no existing one.

 

open <tmName>[ <srcLoc>[ <trgLoc>]]
close[ <tmName>|all]

Closes one or all TM. If no parameter is provided, the command closes the current TM (if one is open).

close all

Closes all open TMs.

close <tmName>

 

create <tmName>[ <srcLoc>[ <trgLoc>]]
add locale <locale>
add field <fieldName>
remove locale <locale>
remove field <fieldName>
set source <srcLoc>
set target <trgLoc>
<alias> is <name>
quit|exit
close[ <tmName>]
show tms
show columns

Displays the list of the fields that are shown when listing entries. To list all the fields of the TM use show fields.

show fields

Displays the list of the fields available in the TM.

show locales

Displays the list of the locales in the current TM.

show aliases
 

Examples

Add a field to a TMX file

J>use memory
J>import myfile.tmx mytm
mytm J>add field newFiledName
mytm J>export myfile.tmx
C:\>jumbo -c "use memory; import myfile.tmx; add myfield; export myfile.tmx; quit"
